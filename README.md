<h3 align="center">vTbaleWidget</h3>

  <p align="center">
    vTableWidget is a custom widget with some basic and necessary functionalities. This widget support loading data from a list of QStringList/QVariantList and also pagination. 
    You can also set date time format for QDateTime objects and decimal points for floating point values when data stored in QVariantList.

## Getting Started

In order to make use of this widget in your applications alongside within your Qt Creator IDE you need to know how to work with Qt Custom Widget.
If you're not familiar with this concept watch some videos about it in my youtube channel @vrfeducation (Persian version) or @ vrfEducationEnglish (English version) and chose Qt Playlist;

<p align="right">(<a href="#readme-top">back to top</a>)</p>



<!-- USAGE EXAMPLES -->
## Usage

If you need to load data stored in some list such as QStringList/QVariantList in a table like widget with some features such as pagination, This widget will help you alot and reduce ammount of code you need to write for some specific tasks.
